## ✨ Características Principales

-   **🌓 Modo Claro / Oscuro Dinámico**: Sistema de temas completo basado en variables CSS que se adapta a las preferencias del usuario.
-   **📱 Diseño Responsivo**: Experiencia optimizada para móviles, tablets y escritorio.
-   **💎 Estética Premium**: Diseño moderno con efectos de glassmorphism, glows dinámicos y micro-animaciones fluidas.
-   **🎞️ Showcases Interactivos**: Carruseles de proyectos integrados con Swiper.js y modales detallados.
-   **📬 Formulario de Contacto Funcional**: Integración con API para comunicación directa.
-   **🚀 Rendimiento Optimizado**: Desarrollado con Astro para una carga ultrarrápida (Zero-JS por defecto donde es posible).

## 🛠️ Tecnologías Utilizadas

-   **Framework**: [Astro](https://astro.build/)
-   **Lenguaje**: TypeScript / JavaScript
-   **Estilos**: Tailwind CSS & Vanilla CSS (Variables Dinámicas)
-   **Librerías**: Swiper.js (Carruseles), Devicon (Iconografía)
-   **Despliegue**: Vercel

## 📁 Estructura del Proyecto

```text
/
├── public/             # Activos estáticos (favicons, etc.)
├── src/
│   ├── assets/         # Imágenes y recursos multimedia
│   ├── components/     # Componentes de Astro (Navbar, Hero, Projects...)
│   ├── layouts/        # Plantilla maestra (Layout.astro)
│   ├── pages/          # Rutas de la aplicación (index.astro, api/...)
│   └── styles/         # Estilos globales
└── package.json        # Dependencias y scripts
```

## 🚀 Comandos

Todos los comandos se ejecutan desde la raíz del proyecto:

| Comando                   | Acción                                           |
| :------------------------ | :----------------------------------------------- |
| `npm install`             | Instala las dependencias                         |
| `npm run dev`             | Inicia el servidor de desarrollo en `localhost:4321`|
| `npm run build`           | Compila el sitio para producción en `./dist/`     |
| `npm run preview`         | Previsualiza la compilación localmente            |
| `npm run astro -- --help` | Muestra la ayuda de la CLI de Astro               |
